당신은 처음부터 완벽한 데스크톱 포스트잇(sticky note) 어플리케이션을 빌드하는 미션을 받았습니다. 아래 지침서에 명시된 기능, 디자인, 데이터베이스 스키마 및 파일 경로 처리 로직을 100% 동일하게 구현해야 합니다.

### 1. 개요 및 구조 개방
- **언어 및 프레임워크**: Python 3.9+ 및 PySide6(Qt for Python).
- **핵심 윈도우 구성**:
  1. **컨트롤 창 (조작 툴바)**: 프로그램 제어 및 설정을 수행하는 얇은 가로형 도구창.
  2. **메모 창 (포스트잇)**: 개별 생성 및 이동, 크기 조절, 접기/펴기, 색상 변경이 가능한 독립된 메모창.
- **크로스 플랫폼 경로 자동 감지**:
  - 패키징(PyInstaller) 여부에 맞춰 실행 디렉토리를 반환하는 `get_app_directory()` 헬퍼를 구현합니다.
  - macOS (Darwin) 환경일 때는 `GomgomPostit.app` 패키지가 존재하는 상위 폴더를 반환합니다.
  - Windows/Linux 환경일 때는 실행 파일(`.exe`)이 존재하는 바로 그 동일한 폴더를 반환합니다.
  - 일반 개발 구동 시에는 스크립트 파일이 위치한 디렉토리를 반환합니다.
- **통합 파일 시스템**:
  - **로컬 SQLite 데이터베이스**: `[실행 디렉토리]/.gomgom_postit.db` 파일로 고정합니다.
  - **통합 환경 설정**: `[실행 디렉토리]/.gomgom_config` JSON 파일로 병합 관리합니다. 키 구조는 `version_mode` (최초 실행 시 기본값 `"local"`) 및 `control_position` (`{"x": int, "y": int}`)를 지닙니다.

### 2. 컨트롤 창 (조작 툴바)
- **비주얼 디자인**: 테두리 없는 Frameless 윈도우 스타일로, 현대적인 고급스러운 다크 슬레이트 톤과 둥근 모서리로 꾸며집니다.
- **브랜드 타이틀 라벨**: 최좌측에 "곰곰메모" 텍스트 라벨을 두며, 오른쪽의 버튼들과 수직 정렬을 맞추기 위해 상단 패딩(`padding-top: 6px`)을 적용합니다.
- **조작 버튼 배치 (좌측부터 우측 순)**:
  1. **▾ (메모 목록 메뉴 버튼)**: 클릭 시 현재 열려있거나 닫혀있는 모든 메모의 제목 리스트를 드롭다운으로 보여줍니다. 각 항목을 클릭하여 숨기거나 띄울 수 있습니다.
  2. **+ (새 메모 생성 버튼)**: 클릭 즉시 일반 텍스트 포스트잇 또는 할일(체크리스트) 포스트잇을 생성합니다.
  3. **⚙ (설정 톱니바퀴 버튼)**: 컨텍스트 메뉴를 팝업합니다.
     - **서버 모드 실행 중일 때**: "로컬 버전으로 실행" 및 "프로그램 종료"
     - **로컬 모드 실행 중일 때**: "서버 버전으로 실행" 및 "프로그램 종료"
- **최초 실행 시 중앙 정렬**: 설정 파일(`.gomgom_config`)에 저장된 창 좌표가 없으면, 메인 화면의 정중앙(Center of screen)에 툴바를 배치합니다. 앱 종료 시 마지막 좌표를 설정 파일에 갱신 기록합니다.

### 3. 두 가지 구동 모드
- **로컬 모드 (최초 실행 시 기본값)**:
  - 로그인 창을 전혀 띄우지 않고 다이렉트로 자동 로그인 처리합니다.
  - 프로그램 내부적으로 기본 계정명을 `"local_user"`로 인식하여 로컬 데이터베이스(`.gomgom_postit.db`)에 메모를 기록합니다.
- **서버 모드**:
  - 전환 시 로그인 인증 다이렉트 창이 실행됩니다.
  - 다음 원격 MariaDB 서버와 연동합니다:
    - **Host**: `[your host ip]`
    - **Port**: `[your database port]`
    - **User**: `[your database user]`
    - **Password**: `[your database password]`
    - **Database**: `[your database name]`
  - 데이터베이스의 원격 접속 자격 증명은 보안 강화를 위해 대칭 XOR 암호화(32바이트 PBKDF2 솔트 조합)한 뒤 Base64 문자열로 소스코드에 삽입하여 노출을 숨겨야 합니다.
  - "자동 로그인"이 체크된 경우 다음 실행 시 로그인 창을 띄우지 않고 자동 복원 로그인(암호화 검증) 처리합니다.

### 4. 로그인 및 계정 관리 창
- **UI 스타일링**: 테두리 없는 프리미엄 슬레이트 테마(`#1E293B`)에 포커스 시 엠버 골드 테두리 효과(`#F59E0B`)를 줍니다.
- **자동 로그인 체크박스**: 체크박스 선택 시, 엠버색 배경 위에 선명한 흰색 V자 모양의 벡터 그래픽(`✓`) 아이콘을 노출합니다. Qt Style Sheets(QSS)에서 인라인 Base64 Data URI를 직접 읽어오지 못하므로, 프로그램 구동 시 `[실행 디렉토리]`에 임시로 `check.svg` 파일(이후 `.gitignore` 등록 처리)을 생성하여 `url("check.svg")` 형식으로 스타일시트에 매핑하도록 구현해야 합니다.
- **아이디 제한**: 사용자 아이디는 반드시 이메일 형식만 허용하며, 정규식(`^[\w\.-]+@[\w\.-]+\.\w+$`)으로 체크합니다.
- **계정 모드 지원**:
  - **로그인**: 자동 로그인 체크박스와 하단 전환 링크를 탑재합니다.
  - **회원가입**: 이메일 아이디와 패스워드를 2번 입력받아 MariaDB `users` 테이블(`username`, `password_hash`, `salt`)에 직접 신규 저장합니다.
  - **비밀번호 변경**: 아이디, 현재 비밀번호, 새 비밀번호를 입력받아 서버 해시 정보를 갱신합니다.
  - **비밀번호 초기화**: 이메일 아이디만 단일 입력받아 일치하는 회원이 존재하면 랜덤 6자리 숫자 비밀번호로 임시 갱신하고, 해당 주소로 Gmail SMTP 보안 전송을 시도합니다.
    - **SMTP Server**: `smtp.gmail.com`
    - **Port**: `465` (SSL 전용)
    - **Username**: `[your gmail user]` (소스코드 내에 XOR + Base64로 암호화 삽입)
    - **Password**: `[your gmail app password]` (소스코드 내에 XOR + Base64로 암호화 삽입)
    - 백업 로그: 메일 전송 데이터 원본을 홈 디렉토리의 `~/.gomgom_email_logs.txt` 파일에 기록하여 오프라인 시에도 확인 가능하게 조치합니다.

### 5. 메모 창 (포스트잇)
- **UI 및 조작**: 테두리 없는 깔끔한 포스트잇 느낌으로 입체적인 그림자 그림자 효과(Drop Shadow)를 지닙니다.
- **동작 방식**:
  - 상단 헤더를 드래그하여 이동하고, 헤더 더블클릭 시 세로 크기를 최소화하여 제목만 노출하는 접기/펴기 상태를 토글합니다.
  - 우클릭 메뉴를 통해 옐로우, 그린, 블루, 퍼플 등의 메모지 색상 전환, 메모 닫기, 완전히 메모를 삭제하는 기능을 지원합니다.
  - **중요 표시(별 아이콘) 버튼**: "항상위" 고정(핀) 버튼 오른쪽에 별 아이콘(`★` / `☆`) 버튼이 위치하며, 클릭 시 중요 상태를 토글합니다.
    - 메모가 중요 상태(별 아이콘 채워짐 `★`)일 때, 우측 상단 닫기(`✕`) 버튼을 누르면 닫기/삭제가 제한되고 "중요 설정(★)된 메시지는 삭제할 수 없습니다." 경고 창을 팝업합니다.
- **메모 유형**:
  - **일반 텍스트**: 자유로운 텍스트 입력 에디터.
  - **할일(To-Do) 목록**: 아이템 생성, 체크 상태 토글, 항목 내용 변경 및 우측 삭제 버튼을 배치한 체크리스트.
- **영속화(저장)**: 메모의 속성(`id`, `username`, `type`, `content`, `todo_items` JSON 변환본, `x`, `y`, `width`, `height`, `color`, `always_on_top`, `is_important`)을 실행 중인 모드(로컬/서버)에 따라 SQLite 또는 MariaDB 테이블에 실시간 자동 업데이트합니다.

---

### 6. 빌드 및 아이콘 패키징
- PyInstaller 명세서(`.spec`)를 통해 각 OS에 맞게 패키징합니다.
  - macOS: `GomgomPostit.app` 패키지로 생성하며 아이콘은 `app_icon.icns`를 연동합니다.
  - Windows: `GomgomPostit.exe`로 생성하며 아이콘은 `app_icon.ico`를 연동합니다.
- 아이콘 모양은 노란색의 스티커 포스트잇 모양에 오른쪽 하단 모서리가 부드럽게 말려 올라간 깔끔한 투명 배경 알파 일러스트를 사용합니다.
